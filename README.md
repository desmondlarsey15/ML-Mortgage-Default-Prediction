# ML-Mortgage-Default-Prediction
# Home Credit Default Risk — End-to-End ML Pipeline

A machine learning project that predicts the probability of a loan applicant defaulting on their repayment, built on the [Home Credit Default Risk](https://www.kaggle.com/c/home-credit-default-risk) dataset from Kaggle.

---

## Project Overview

Many people struggle to get loans due to insufficient or non-existent credit histories. Home Credit uses a variety of alternative data to predict client repayment abilities. This project builds and compares multiple ML models to predict default risk, with a focus on maximizing ROC-AUC.


## Data Source

[Home Credit Default Risk — Kaggle Competition](https://www.kaggle.com/c/home-credit-default-risk)
